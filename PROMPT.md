## 1. プロジェクト概要

地方（新潟・津南エリア）発のペットライフスタイルブランド **「White Snow Paws」** のオンラインショップを作成してください。
雪国のクリーンなイメージと天然木のぬくもりを融合させたデザインが特徴です。

### ブランドコンセプト
**「雪のように純粋な愛情を、あなたの手から愛犬へ」**

新潟・津南の雪国で、ひとつひとつ手作りするBioThane（バイオシーン）素材のリード＆カラー。
大手ブランドでは5,000円前後の価格帯ですが、一つずつ丁寧に手作りすることで高品質＆リーズナブルを実現。
防水・軽量・お手入れ簡単なBioThane素材で、雪国のアクティブな犬との暮らしにぴったり。

### 商品カテゴリ

#### A. BioThane（バイオシーン）リード & カラー 【メイン商品】
- **BioThane素材**を使用した防水・軽量・高耐久のリード・カラー（首輪）
- 一つ一つ**手作り**で工房から直接お届けするため、リーズナブルな価格を実現
- カラーバリエーション豊富（カスタムオーダー対応）
- 雪・雨・泥にも強い素材でアウトドア派に最適
- 商品ラインナップ: リード / カラー（首輪）/ ハーネス / リードストラップ / セット商品

#### B. オーダーメイド木工品（ウッドバーニング）【検討中】
- 地元の職人が**手焼き**で制作する一点物
- 誕生日祝い・メモリアル（虹の橋）用の特別感ある商品
- フードボウルスタンド、ネームプレート、メモリアルフレーム等
- **「世界にひとつだけ」**の訴求が重要
- ※ 現在企画検討中のため、トップページではティザー的に紹介

#### C. こだわりのジビエフード（OEM）【検討中】
- **鹿肉・熊肉・津南ポーク**等の地元素材を使用
- 健康志向・アレルギー対応の安全なペットフード
- 無添加・国産・トレーサビリティを訴求
- ※ 現在企画検討中のため、トップページではティザー的に紹介

### 技術スタック
- **HTML5 / CSS3 / JavaScript**（バニラ）
- **Swiper.js** — カルーセル/スライダー
- **Google Fonts** — 日本語＋英語の組み合わせ
- **モバイルファースト**設計（Instagramストーリー広告からの流入を想定）
- GitHub Pages でホスト可能な静的サイト

## 2. 全体デザイン方針

### コンセプト: 「雪国の清潔感 × 天然木のぬくもり × アクティブなアウトドア」

- **ダークウッド〜ミディアムウッドの木目テクスチャ**を前面に押し出したデザイン
- ヘリンボーンパターンだけでなく、木板（プランク）風、丸太断面（年輪）、ログハウス風のテクスチャも活用
- セクション背景に木目テクスチャのパターンをより大胆に使用
- ホワイト（雪）× ブラウン（木）× グリーン（森）の3トーン構成
- BioThaneのカラフルなリード商品が映える、落ち着いた木調の背景

### カラースキーム
| 用途 | カラー |
|------|--------|
| 背景（メイン） | `#FAF8F5`（スノーホワイト） |
| 背景（ウッド系セクション） | `#E8DDD0`（ライトウッド） |
| 背景（ダークセクション） | `#3C3228`（ダークウッド） |
| テキスト（メイン） | `#2C241C`（ディープブラウン） |
| テキスト（サブ） | `#7A6B5D`（ミディアムブラウン） |
| テキスト（白背景用） | `#FAF8F5`（スノーホワイト） |
| アクセント（主） | `#4A7C59`（フォレストグリーン） |
| アクセント（副） | `#C4956A`（テラコッタ／暖色ウッド） |
| ボタン/CTA | `#4A7C59` → ホバー `#3A6349` |
| ウッド装飾（明るめ） | `#D4C4B0`（ナチュラルウッド） |
| ウッド装飾（暗め） | `#8B6F47`（ウォールナット） |
| 木目ボーダー | `#A08060`（オーク調） |

### タイポグラフィ
- **英語見出し**: `"Playfair Display"` or `"Cormorant Garamond"` — クラフト感のあるセリフ体
- **日本語見出し**: `"Zen Old Mincho"` or `"Noto Serif JP"` — 温かみのある明朝体
- **日本語本文**: `"Noto Sans JP"` — Regular/Medium
- **サイズ感**: ヒーロー見出し `40-56px` / セクション見出し `24-32px` / 本文 `14-16px`

### 全体トーン
- **ナチュラル・ウッディ・アウトドア・クリーン**
- 余白を贅沢に使う（セクション間 `80-120px`）
- 木目テクスチャを装飾だけでなく、**カードの背景やセクション全体の背景に大胆に使用**
- 写真が主役 — BioThaneリードのカラフルさ、犬との自然の中のアクティブシーン
- ウッドテクスチャ + スノーホワイトのコントラストで温もりと清潔感を両立
- **カード・ボタンに木のフレーム感（木枠風ボーダー）**を活用

### ウッド調デザインの具体的な強化ポイント
1. **ヘッダー**: ダークウッドの木目テクスチャ背景
2. **セクション区切り**: ヘリンボーンパターンのディバイダーに加え、木板の境目風の装飾
3. **カード**: 木枠風のボーダー（`border-image` or box-shadow で木のフレーム感を表現）
4. **フッター**: ダークウッド + ログハウス風の雰囲気
5. **ボタン**: 角丸の木のタグ（ネームタグ）風デザイン
6. **背景パターン**: CSSで年輪や木の繊維を連想させる微細パターン
7. **セクション見出し**: 木の焼印風のテキスト装飾

## 4. アニメーション & インタラクション

### スクロールアニメーション
```css
/* フェードアップ（基本） */
.fade-up {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}
.fade-up.active {
  opacity: 1;
  transform: translateY(0);
}
```
- **Intersection Observer API** でスクロール検知
- 各カードは `0.1s` ずつディレイをずらす（スタガー効果）

### ホバーエフェクト
- **商品画像**: 2枚目画像にクロスフェード切替（カラーバリエーション表示）
- **カテゴリカード**: 画像 `scale(1.05)` + ウッドフレーム風ボーダー出現
- **ボタン**: 背景色グラデーションスライド + 矢印アイコン移動
- **リンク**: 下線がスライドイン（アクセントカラー）

### ヘッダー
- スクロール `100px` 以降で半透明ウッド背景 + ボックスシャドウ
- ロゴサイズ縮小トランジション

### ウッド調装飾の実装方法
- **CSS背景パターン**: 木目テクスチャ画像 or CSSグラデーションで木目再現
- **セクション区切り**: ヘリンボーンパターン + 木板の継ぎ目風ライン
- **カードフレーム**: `border` + `box-shadow` の組み合わせで木枠感を表現
- **テクスチャオーバーレイ**: `::before`/`::after` 疑似要素で木目テクスチャを重ねる
- **焼印風テキスト**: `text-shadow` + 若干のブラウン系グラデーションでウッドバーニング風

## 5. モバイルファースト & SNS連携設計

### Instagram広告 → サイト流入を最適化
- **ファーストビュー**: スマホで見た瞬間にBioThaneリードの魅力が伝わる構成
- **CTAボタン**: 親指が届く画面下部に固定フローティングボタン（「カートに入れる」or「オーダーする」）
- **ページ読み込み速度**: 3秒以内（画像最適化、遅延読み込み必須）
- **スクロール導線**: 縦スクロールだけで完結、横スワイプは補助的

### レスポンシブブレークポイント
| ブレークポイント | デバイス | グリッド列数 |
|---------------|---------|------------|
| `~767px` | スマートフォン | 1-2列 |
| `768px~1023px` | タブレット | 2-3列 |
| `1024px~` | デスクトップ | 3-4列 |

- モバイルファースト設計（`min-width` メディアクエリ）
- タッチデバイスではホバー効果を無効化
- スライダーはスワイプ対応
- **スマホ表示の確認を最優先**
