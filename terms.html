第1条（会員登録）

  1. 本サービスへの会員登録は無料です。登録情報は正確かつ最新の情報を入力してください。
  2. 虚偽の情報を登録した場合、当店は予告なく会員資格を取り消すことができます。
  3. 登録したメールアドレスとパスワードはお客様自身の責任で管理してください。第三者への譲渡・貸与は禁止します。
  4. アカウントの不正利用による損害について、当店は責任を負いません。
  5. 退会をご希望の場合はお問い合わせページよりご連絡ください。

第2条（ご注文・購入）

  1. ご注文はウェブサイト上のフォームまたはLINEにてお受けします。
  2. カスタムオーダー商品はご注文確定後、原則としてキャンセルできません。
  3. 在庫状況や素材の仕入れ状況によって、ご希望に添えない場合があります。その際はご注文後にご連絡します。
  4. 当店からの注文確認メールをもってご注文の受付完了とします。

第3条（料金・お支払い）

  1. 商品価格はすべて税込み表示です。
  2. 送料は全国一律550円（税込）です。¥5,000以上のご注文で送料無料となります。
  3. お支払い方法は、クレジットカード・銀行振込・コンビニ払い等からお選びいただけます（決済方法は順次追加予定）。
  4. 銀行振込の場合、ご注文から5営業日以内にお振込みください。期限内にご入金が確認できない場合、ご注文をキャンセルさせていただく場合があります。

第4条（配送・お届け）

  1. カスタムオーダー商品の製作期間は、ご注文確定から約1週間を目安としています（混雑時は変動する場合があります）。
  2. 在庫品は入金確認後、1〜3営業日以内に発送します。
  3. お届けは日本国内に限ります（海外発送は現在対応しておりません）。
  4. 天候・交通事情・配送業者の都合により、お届けが遅れる場合があります。
  5. お届け先の住所が誤っていた場合の再送費用はお客様のご負担となります。

第5条（返品・交換）

  1. 商品到着後7日以内に限り、以下の場合に限って返品・交換をお受けします。
    • 商品に製造上の不具合・破損があった場合
    • ご注文と異なる商品が届いた場合
  2. カスタムオーダー商品（色・サイズをご指定いただいたもの）は、不具合がない限り返品・交換をお受けできません。
  3. 返品・交換をご希望の場合は、まずお問い合わせページまたはLINEにてご連絡ください。返品方法・送料についてご案内します。
  4. 不具合による返品・交換の送料は当店が負担します。

第6条（キャンセル）

  1. 在庫品のご注文は、発送前であればキャンセルが可能です。発送後はキャンセルをお受けできません。
  2. カスタムオーダー商品は、製作開始前（ご注文確定から24時間以内）に限りキャンセルが可能です。製作開始後はキャンセルをお受けできません。
  3. キャンセルのご連絡はお問い合わせページまたはLINEにてお願いします。

第7条（禁止事項）

本サービスのご利用にあたり、以下の行為を禁止します。

  • 虚偽の情報を登録・提供する行為

  • 当店のサービス運営を妨げる行為

  • 不正アクセス、システムへの不当な負荷をかける行為
  • 法令または公序良俗に反する行為
